# Calculator App

This is a simple calculator web app that I built using HTML, CSS and JavaScript.  
The goal of this project was to practice JavaScript logic, event handling, and working with user input.

---

## Features

- Basic arithmetic operations:
  - Addition (+)
  - Subtraction (−)
  - Multiplication (×)
  - Division (÷)
- Clear button to reset the calculator
- Responsive layout (works on different screen sizes)
- Real-time display updates when buttons are clicked

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript

---

## What I Learned

While building this project, I improved my understanding of:

- DOM manipulation (selecting elements and updating the display)
- Event listeners (handling button clicks)
- Working with strings and numbers in JavaScript
- Basic debugging when calculations didn’t work as expected
- Structuring small projects properly

---

## Challenges

Some of the things I struggled with:

- Making sure the calculator doesn’t break when clicking operators multiple times
- Handling decimal numbers correctly
- Resetting the calculator state properly after pressing "="

I had to test different cases and fix bugs step by step.

## Future Improvements

If I had more time, I would:
